# HttpContext Learning – ASP.NET Core 9

This repository contains exercises, notes, and sample code from **Frank Liu’s ASP.NET Core [.NET 9] Deep Dive course on Udemy**, focused on understanding and working with `HttpContext`.

## 📚 Purpose
The goal of this repo is to:
- Explore the internals of `HttpContext` in ASP.NET Core 9.
- Practice handling requests, responses, cookies, sessions, and middleware.
- Document key learnings from the course in a structured way.
- Provide runnable examples for future reference.

## 🛠️ Topics Covered
- Lifecycle of `HttpContext` in ASP.NET Core.
- Accessing request and response objects.
- Working with headers, query strings, and form data.
- Managing cookies and session state.
- Dependency injection and service lifetimes in relation to `HttpContext`.
- Middleware and pipeline behavior.
- Common pitfalls and best practices.
